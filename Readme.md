# Linux Network Security Lab (Snort IDS + Firewall + hping)

This project demonstrates the deployment of a small network security lab using VirtualBox. The environment consists of two Linux virtual machines connected over an isolated internal network.

The lab focuses on configuring static IP addressing, deploying Snort as an Intrusion Detection System (IDS), configuring firewall rules, and generating network traffic using hping3 to simulate attacks and validate IDS detection.

## Objectives

* Deploy an isolated VirtualBox network
* Configure static IP addressing
* Install and configure Snort IDS
* Configure Linux firewall rules
* Generate ICMP, TCP and UDP traffic using hping
* Analyse alerts generated by Snort

## Lab Environment

### Virtual Machines

VM2    Attack Client

### Network

```text

```

### IP Addressing

```text
VM1
192.168.1.1

VM2
192.168.1.2
```

## Workflow

1. Configure the VirtualBox internal network.
2. Assign static IP addresses.
3. Install Snort.
4. Configure Snort rules.
5. Configure firewall rules.
6. Generate traffic using hping.
7. Monitor and analyse Snort alerts.
